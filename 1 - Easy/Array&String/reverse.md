Given an array (or string) A[] of length x, the task is to reverse the array/string.

Input format:
```
x
a[0] a[1] a[2] ... a[x-1]
```
Note: if the input is a string, there is no space between values.

## Example:

Input:
```
3
1 2 3
```
Output: 
```
3 2 1
```

## Example 2:

Input 2:
```
5
Hello
```
Output 2:
```
olleH
```
